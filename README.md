# PhoneDork

A Chrome extension enabling a user to select a phone number in a web page, and
quickly execute searches with:

* Google
* Facebook
* Twillio

The Google search checks for the following number formats:

* XXXXXXXXXX
* (XXX) XXX-XXXX
* (XXX) XXX XXXX
* XXX.XXX.XXXX
* XXX-XXX-XXXX
* XXX XXX XXXX
